## Requisitos

Este aplicativo requer Ruby v2.5 ou superior e Rails 6.0 ou superior.

## Instalação

```bash
$ bundle install
$ yarn install
$ rake db:migrate
```

Por último, renomeie o arquivo `.env.sample` para` .env`.
